# [Рус]

Проект реверс-инжиниринга GTA 3, re3, включал в себя русскую локализацию от Бука, однако из-за остановки разработки проекта русская локализация не была полностью завершена — в игре оставались непереведенные новые строки и ошибки базовой локализации. Этот репозиторий имеет целью завершить русскую локализацию проекта re3.

Названия автомобилей были приведены к общему стилю. Так, раньше часть названий были просто переведены (например, Bobcat — Рысь) или записаны транслитом (на мой взгляд, это правильный вариант), то сейчас все транспортные средства записаны транслитом, так как это было сделано в локализации от 1С (Софт-клаб).

# [Eng]

The GTA 3 reverse engineering project, re3, included Russian localisation by Buk, but due to the project being halted, the Russian localisation was not fully completed — the game still contained untranslated new lines and errors in the basic localisation. The purpose of this repository is to complete the Russian localisation of the re3 project.

The names of the cars have been brought into line with the overall style. Previously, some of the names were simply translated (e.g. Bobcat — Lynx) or written in transliteration (in my opinion, this is the correct option), but now all vehicles are written in transliteration, as was done in the localisation by 1C (Soft-Club).
